# Codeforces Clone

A frontend clone of the popular competitive programming platform Codeforces built using pure HTML and CSS.

This project recreates the layout and UI of the Codeforces homepage including:

- Navigation bar
- Contest announcement section
- Sidebar widgets
- Top rated users
- Top contributors
- User profile card
- Find user section

Built mainly for improving:
- Flexbox understanding
- Layout structuring
- Responsive thinking
- UI recreation skills

---

## 🚀 Features

- Fully structured homepage layout
- Sidebar cards like original Codeforces
- Styled tables for rankings
- User profile section
- Sponsor section
- Custom buttons and forms
- Flexbox-based layout
